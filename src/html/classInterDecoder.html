Decodes a frame from the file encoded using block-based motion compensation (inter-frame encoding). Divides the frame into blocks of size block_size x block_size, decodes each block summing the prediction value (old_frame's block) and the error value (encoded block's value). Decoded data is stored in curr_frame Mat.
