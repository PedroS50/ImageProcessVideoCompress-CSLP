Detailed Description

Encoder which utilizes both interframe and intraframe encoding based on Golomb codes.

File used to demonstrate a hybrid encoder which encodes a video file, storing it on the computer so that it can later decoded. Also demonstrates decoding, showing the decoded frames while decoding.
